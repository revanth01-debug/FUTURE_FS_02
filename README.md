# CRM Website

A modern Customer Relationship Management (CRM) web application built using Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript. The application helps users manage leads, track client information, and organize follow-ups efficiently.

## Features

* User Registration and Login
* Lead Capture and Management
* Client Information Storage
* Lead Status Tracking
* Notes and Follow-up Management
* Responsive User Interface

## Run Locally

1. Install dependencies

```bash
npm install
```

2. Start the server

```bash
npm start
```

3. Open in browser

```text
http://localhost:3000
```
